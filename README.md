## ✨ 项目介绍

这是一个独特的爱情纪念网站，记录了两位主人公从 **2026 年 4 月 14 日** 开始的 30 天星轨之旅。

> 一条临时改道的1号线，把八宝山和玉泉路轻轻推近；一顿晚饭，把两个原本陌生的人接进同一条星轨。

项目以北京地铁 **1 号线** 为背景，记录了每一站的故事、对话和美好回忆，最终驶向爱情的终点——**520**。

## 🚀 快速开始

### 📋 前提要求

- 任何现代浏览器（Chrome、Firefox、Safari、Edge）
- 基本的 Web 服务器（用于提供静态文件）

### 💻 本地运行

1. **克隆项目**
   ```bash
   git clone https://github.com/BothCats/star-rail-520.git
   cd star-rail-520
   ```

2. **启动本地服务器**
   
   **使用 Python 3:**
   ```bash
   python -m http.server 8000
   ```
   
   **使用 Node.js (http-server):**
   ```bash
   npx http-server
   ```
   
   **使用 Live Server (VS Code):**
   - 安装 "Live Server" 扩展
   - 右键 `index.html` 选择 "Open with Live Server"

3. **访问网站**
   ```
   打开浏览器访问：http://localhost:8000
   ```
